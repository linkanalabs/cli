# CLAUDE.local.md — regras locais do projeto

## Specs / planos não vão pro repo

- **Nunca commitar** arquivos em `docs/superpowers/specs/` nem
  `docs/superpowers/plans/`. São artefatos de planejamento local (superpowers),
  não pertencem ao repo versionado.
- Se aparecerem staged num diff, remover do stage antes de commitar.
